# ILAW – Smart Home Lighting System

ILAW is a Flutter mobile application for controlling smart home lighting, 
built with Firebase and integrated with ESP32 hardware. Users can remotely 
toggle lights, set automated on/off schedules, and manage their devices 
through a clean, intuitive interface.

# Features
- 🔐 **Secure Authentication**– Email/password sign-up, login, and password 
  reset powered by Firebase Auth
- 💡 **Remote Light Control** – Toggle connected lights in real time via 
  Firebase Realtime Database
- ⏰ **Smart Scheduling**- Set automatic on/off times with timezone-aware 
  scheduling
- 🤖 **AI Assistant** – Built-in assistant module for smart lighting insights
